# SnakeGame

A simple snake game made for fun that i play it some times in my free time :)

### Install Dependencies

```bash
sudo zypper in SDL2 SDL2-devel libSDL2_ttf-2_0-0 SDL2_ttf-devel
```

### Run
```
cargo run
```
